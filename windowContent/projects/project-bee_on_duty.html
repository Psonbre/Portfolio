Bee On Duty

December 2023

Description

Bee on Duty is a game developed in 48 hours during the 2023 Valleyfield Intercollegiate Game Jam. The main objective of the game is to protect a beehive. Players must build defenses using pollen collected from flowers to safeguard their hive from external threats.
