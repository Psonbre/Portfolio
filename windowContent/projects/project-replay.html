Re: Play

June 2, 2024

Description

A game made in 48 hours for the "Micro Jam 15: Ghost". The theme "ghost" was interpreted by using a "replay ghost" as the main character. The goal of the game is to imitate the "player" as precisely as possible — sometimes following a path that’s complex to replicate.

Additional Information

  • "Best Concept" award
  • Developed with Unity
  • Published on Itch.io
