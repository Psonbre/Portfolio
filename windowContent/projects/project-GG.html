Goblin's Greed

Juillet 2023

Description

Goblin's Greed est un mélange entre un hack and slash vu de dessus et un tower defense. Le but est d'augmenter la taille de votre trésor en le protégeant des héros qui tenteront de le voler. Vous pouvez remplir votre donjon de pièges ou améliorer vos capacités de combat pour les affronter dans un combat équitable.

Information additionnelle

  • Développé seul
  • Développé en 48h pour la GMTK Game Jam 2023 avec le thème "Roles reversed"
  • Donjon généré aléatoirement
  • Page Itch.io du jeu
