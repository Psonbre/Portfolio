Greening

March 2021

Description

Greening est un jeu développé en 48 heures pour la MothoGlacia Construct Reddit GameJam. Ce projet est un jeu où le joueur doit combattre un boss prenant l'apparence d'une flamme. Le but est d'éteindre la flamme avant qu'elle ne consume entièrement la végétation environnante.
