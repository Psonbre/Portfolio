Greening

March 2021

Description

Greening is a game developed in 48 hours for the MothoGlacia Construct Reddit GameJam. This project is a game where the player must fight a boss that takes the form of a flame. The goal is to extinguish the flame before it completely consumes the surrounding vegetation.
