Blue Light

Juillet 2024

Description

Blue Light est un jeu développé sur Unity en 48 heures pour la Kenney Jam 2024, avec le thème « connections ». Dans ce jeu le joueur contrôle un robot qui ne doit pas s'éloigner de ses antènes, à risque de se faire déconnecter.
