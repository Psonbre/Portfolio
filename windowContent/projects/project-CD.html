Collateral Damage

février - avril 2024

Description

Une application / site web qui facilite la création et l'édition de scénarios interactifs "dont vous êtes le héros" en réalité virtuelle. Les utilisateurs peuvent importer leurs propres vidéos 360° et les connecter visuellement grâce à un éditeur intuitif. Il est également possible d'intégrer des questions à choix multiples, permettant ainsi de créer des embranchements dans les scénarios.

Une fois le scénario complété, il est possible de l'exporter sous forme d'APK pour Quest 2/3, d'exécutable Windows, ou de le visionner directement dans un navigateur web.

Information additionnelle

  • Écrit en JavaScript et CSS pur
