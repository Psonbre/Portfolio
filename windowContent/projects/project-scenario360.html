360 Scenario Editor

June 2024

Description

A web app that simplifies the creation and editing of interactive "choose-your-own-adventure" scenarios in virtual reality. Users can import their own 360° videos and visually connect them through an intuitive editor. It's also possible to add multiple-choice questions, enabling branching storylines.

Once the scenario is complete, it can be exported as an APK for Quest 2/3, a Windows executable, or viewed directly in a web browser.

Additional Information

  • Developed solo
  • Written in pure JavaScript and CSS
  • 360° viewer developed in Unity
